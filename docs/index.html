MYCAM Virtual Camera for Android

Use local media or OBS RTMP streams as your camera source on a rooted Android phone with LSPosed. MYCAM supports Watch mode, Clone mode, Android ID changer, remote controls, MediaMTX RTMP tools, WhatsApp, Messenger and other supported camera applications.

Features

Virtual camera source

Use selected local media as a camera feed in supported Android camera applications.

Live RTMP from OBS

Stream from OBS to the local MediaMTX server and activate Live RTMP inside MYCAM.

Watch mode

Keep the selected source ready for the target application when its camera opens.

Remote controls

Use MYCAM controls to manage the virtual camera workflow while the target app is active.

Clone mode

Create separated supported social app profiles for accounts you own, with MYCAM controls available per target.

Android ID changer

Change Android ID profiles for testing, app-profile isolation, and supported cloned app environments.

App support

Works with WhatsApp, Messenger and supported applications that open the Android camera.

Root and LSPosed

Built for rooted Android phones with LSPosed enabled for the selected target app.
